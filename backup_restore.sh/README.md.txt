# Backup and Restore Script

Este es un script en Bash que automatiza la creación de copias de seguridad y su restauración.

## Características
- Comprime archivos o carpetas especificadas.
- Guarda los backups en un directorio con marcas de tiempo.
- Permite restaurar cualquier backup a una ubicación personalizada.

## Cómo usar
1. Descarga el script:
   ```bash
   wget https://github.com/tu-usuario/portafolio/blob/main/backup_restore.sh

Hazlo ejecutable:

chmod +x backup_restore.sh

Ejecuta el script:

./backup_restore.sh

Ejemplo de Salida

Menú del script:

Seleccione una opción:
1. Realizar Backup
2. Restaurar Backup
3. Salir

Backup generado:

Backup completado: ./backups/backup_20241212153000.tar.gz
